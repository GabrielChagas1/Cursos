## 🎯 Objetivo do Projeto

Trabalhar com uma api e trazer dados de um pokemon específico

### 🦂 Exemplos de Pokemon  
 * limber
 * ditto
 * quick-powder
 * pikachu

### 🚀 Como rodar o projeto

Siga os passos abaixo
```bash
# Install the dependencies
$ npm install

# Start the project
$ node index.js

# choose a house
$  Digite o nome de um pokemon: 
```
